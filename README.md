# Nilesh Barandwal — Portfolio

Personal portfolio site. Built with React, TypeScript, and Vite. Dark cinematic design.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v3
- Geist + Cabinet Grotesk fonts

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Deploys to GitHub Pages at `https://NileshBarandwal.github.io/portfolio/`.
